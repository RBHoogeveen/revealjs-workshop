Before cloning the repository, make sure you already install NodeJS (10.0.0), then checkout your branch 

						$ git clone git@github.com:RBHoogeveen/revealjs-workshop.git
						$ git checkout groups/group-{your-group-number}
Go to the project's folder and install packages needed. This might take a bit! :) 

						$ cd revealjs-workshop && npm i
Start your application and view your slides at http://localhost:8080 

						npm start
For documentation, you can find it at https://revealjs.com.
